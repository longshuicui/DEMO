# 代码库索引

## 如何构建索引
1. 哪些信息需要构建索引：这里涉及到使用的字段有哪些
   * 对于一个代码仓库来讲，不同文件依赖是一个重要的信息
   * README文件可能包括代码功能介绍、使用说明、背景等信息，可以看成一个摘要
   * 具体的内容/代码块，代码块要不要语法解析？
2. 分块逻辑
   * 文件分块
   * 内容分块
3. 怎么检索 
   * Embedding检索：embedding模型选择，考虑向量维度、HitRate、MRR
   * BM25检索：如何使用BM25
   * 关键词检索：如何构建关键词索引
4. 怎么存储
   * Embedding如何存储，需结合内存占比考虑，数据量小，维度小可以直接放内存
   * 好像也只能放内存

## 索引如何更新
1. 整体代码发生改变，需要重建索引
2. 部份代码发生改变，仅更新修改部份索引，如何动态的处理
